Easily toggle the display of tax in product descriptions with a simple shortcode.

== Description ==

WooCommerce Tax Toggle is a simple yet powerful plugin that allows your customers to toggle between viewing product descriptions with or without tax. With just a single shortcode, you can enhance the user experience by providing a clear and dynamic display of product pricing.

= Features =
* Add a toggle button for tax in product descriptions.
* Shortcode `[tax_toggle]` to easily integrate into your WooCommerce site.
* Dynamically update the product description to show prices with and without tax.
* Lightweight and easy to use.

= Usage =
1. Install and activate the plugin.
2. Add the `[tax_toggle]` shortcode to your product descriptions or any other place where shortcodes are supported.
3. The toggle button will appear, allowing users to switch between viewing product descriptions with or without tax.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-tax-toggle` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `[tax_toggle]` shortcode in your product descriptions.
